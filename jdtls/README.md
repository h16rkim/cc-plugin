# jdtls

Eclipse JDT Language Server for Java support in Claude Code.

## Installation

### macOS (Homebrew)

```bash
brew install jdtls
```

### Verify Installation

```bash
jdtls --version
```

## Supported File Extensions

- `.java` - Java

## Requirements

- JDK 11 or higher
- Gradle or Maven project (for proper project analysis)

## Features

- Code completion
- Go to definition
- Find references
- Hover documentation
- Code diagnostics
